5 Multiple Regression

Multiple linear regression, or simply multiple regression, is a powerful technique that can handle (a) assorted types of independent variables (IVs), i.e., ditchotomous, nominal, continuous, and (b) complex relations among IVs, e.g., nonlinear reglations and interaction effects.

5.1 Multiple Regression Model - The Base Form

In simple linear regression model, we use only one independent variable X to predict Y and the prediction model has two parameters, i.e. the intercept and the slope. In this chapter, instead of usign one, we will use p independent variables, i.e. \(X_1, X_2, \dots, X_p\), to predict Y. As a result, with the population parameters in place, the multiple linear regression model is

\[ Y_i = \mu_i + \varepsilon_i = \beta_0 + \beta_1X_{1i} + \beta_2X_{2i} + \cdots + \beta_pX_{pi} + \varepsilon_i \] where \(\mu_i \equiv E(Y_i|\boldsymbol{X}_i)\) is the deterministric part of the model, referred to as the mean response model or the prediction model.

\[ \mu_i = \beta_0 + \beta_1X_{1i} + \beta_2X_{2i} + \cdots + \beta_pX_{pi} \] and \(\varepsilon_i\) is called the error and

\[\varepsilon_i \overset{iid}{\text{~}} N(0,\sigma^2)\]

The parameters of this linear model are \(\beta_0, \beta_1, \dots, \beta_p\) and \(\sigma\).

For the sake of simplicity, we will introduce vectorized notations. By convention, we will use bold type or an over-head \(\rightarrow\) to indicate a vector, e.g., \(\boldsymbol{v}\) or \(\overset{\rightarrow}{v}\). Let \(\boldsymbol{b} = [b_1,b_2,\dots,b_p]\)

or using the sample parameters and simplified notations

\[ \hat{Y}_i = b_0 + b_1X_{1i} + b_2X_{2i} + \cdots + b_pX_{pi} \]

5.1.1 Multiple Linear Regression Model

\[ Y_i = \hat{Y}_i + E_i = b_0 + b_1X_{1i} + b_2X_{2i} + \cdots + b_pX_{pi} + E_i \] for \(i = 1,2,\dots,n\).

5.2 The Ordinary Least-Squres Method

Similar to the case of simple linear regression, the ordinary least squares (OLS) method can also be applied to estimate parameters for multiple linear regression.

\[ SSR = \sum_{i=1}^n R_i^2 = \sum_{i=1}^n (Y_i-\hat{Y}_i)^2 = \sum_{i=1}^n (Y_i - (b_0 + b_1X_{1i} + b_2X_{2i} + \cdots + b_pX_{pi}))^2 \]

The solutions for \(b_i\), \(i = 1,2,\dots,p\), should minimize \(SSR\), hence the name least-squares method.

5.2.1 Residual Variance

The residual variance (aka conditional variance) is given as follows.

\[ s^2 = MSR = \frac{\sum R_i^2}{df} = \frac{\sum_{i=1}^n (Y_i-\hat{Y_i})^2}{df} = \frac{\sum_{i=1}^n (Y_i - (b_0 + b_1X_{1i} + b_2X_{2i} + \cdots + b_pX_{pi}))^2}{n-(1+p)} \]

It should be clear that \(1+p\) is the number of parameters required to estimate \(\hat{Y}_i\) (i.e., the degrees of freedom we have to spend or give up for fitting the model), which equals \(p\) slope parameters plus 1 for the intercept. The denominator \(n-(1+p)\) is the degrees of freedom that are left for us after fitting the model.

5.2.2 \(R^2\) and ANOVA for Multiple Regression

When dealing with simple linear regression, we have already studied how to derive the ANOVA table, compute the \(R^2\) and perform the \(F\) test. The same results also hold here, which will not be repeated except for one point. The formula for computing \(R^2\) is

\[ R^2 = \frac{\text{SSM}}{\text{SST}} = \frac{ \sum(\hat{Y}_i-\bar{Y})^2 }{ \sum(Y_i-\bar{Y})^2 } \]

The square root of \(R^2\) in the context of multiple regression is called the multiple correlation coefficient, which interestingly enough is the correlation between observed values \(Y_i\) and predicted values \(\hat{Y}_i\).

R_Proof: Multiple Correlation Coefficient

tdat = na.omit(xdat[c("mid3","satmath","satreading","gpa")])
mod1 = lm(mid3 ~ satmath + satreading + gpa, data=tdat)
yhat = predict(mod1)
ybar = mean(tdat$mid3, na.rm=T)
ssm = sum((yhat-ybar)^2, na.rm=T)
sst = sum((tdat$mid3-ybar)^2, na.rm=T)
r2 = ssm/sst
r = cor(tdat$mid3, yhat)
c(sqrt(r2), r)
## [1] 0.07316646 0.07316646
